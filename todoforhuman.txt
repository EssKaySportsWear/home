# EssKay Sportswear - Firebase Rules

## IMPORTANT: Copy ONLY the JSON below (lines between the ``` markers) into Firebase Console → Realtime Database → Rules → Paste → Publish

```json
{
  "rules": {
    "users": {
      ".read": "root.child('users').child(auth.uid).child('email').val() === 'business.esskaysportswear@gmail.com'",
      "$uid": {
        ".read": "$uid === auth.uid",
        ".write": "$uid === auth.uid || root.child('users').child(auth.uid).child('email').val() === 'business.esskaysportswear@gmail.com'"
      }
    },
    "products": {
      ".read": true,
      ".write": "auth != null"
    },
    "carts": {
      "$uid": {
        ".read": "$uid === auth.uid",
        ".write": "$uid === auth.uid"
      }
    },
    "customOrders": {
      ".indexOn": ["userId", "price", "status"],
      ".read": "auth != null",
      ".write": "auth != null"
    },
    "standardOrders": {
      ".indexOn": ["userId"],
      ".read": "auth != null",
      ".write": "auth != null"
    }
  }
}
```

## What each rule does

| Node | Read | Write | Why |
|------|------|-------|-----|
| users | Admin only (list all) | Own profile + Admin | auth.js creates profile on signup, admin manages users |
| users/$uid | Own profile | Own profile + Admin | auth.js reads own profile, admin edits/deletes |
| products | Everyone | Any logged-in user | Product listing is public; checkout decrements stock |
| carts/$uid | Own cart only | Own cart only | Each user manages their own cart |
| customOrders | Any logged-in user | Any logged-in user | orders.html queries by userId; index enables the query |
| standardOrders | Any logged-in user | Any logged-in user | orders.html queries by userId; index enables the query |

## Database paths used in code

- **auth.js**: `users/{uid}` — read/write own profile
- **index.html**: `products` (read), `carts/{uid}` (read/write), `customOrders` (push)
- **cart.html**: `carts/{uid}` (read/write/delete), `standardOrders` (push), `products/{id}` (read + write quantity)
- **orders.html**: `customOrders` (query by userId), `standardOrders` (query by userId), cancel = delete order
- **admin.html**: `users` (read all), `customOrders` (read all, update price/status), `products` (CRUD)
