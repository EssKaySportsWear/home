# EssKay Sportswear

E-commerce site for sportswear with Firebase backend.

## Pages
- index.html - Home (hero with custom order)
- products.html - Products page
- cart.html - Shopping cart
- login.html - Login/SignUp

## Navbar
Glassmorphism design with icons and active indicator animation.

## Run
Open index.html in browser.

## Tech
- HTML/CSS/JS
- Firebase
- Google Fonts (Sora, Bebas Neue)
